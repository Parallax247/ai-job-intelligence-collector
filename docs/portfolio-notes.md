# Portfolio presentation notes

Recommended five-minute walkthrough:

1. Explain the repeated job-search problem and the decision to build a local-first collector.
2. Show the Streamlit interface and reusable keyword library.
3. Start a small run with two keywords and two jobs per keyword.
4. Open the generated Excel workbook and screenshot directory.
5. Show the adapter boundary in `scrapers/base.py` and one platform implementation.
6. Close with the test suite and the planned scoring / GPT-analysis layer.

Suggested repository description:

> Local-first job intelligence collector using Streamlit, Playwright/CDP, platform adapters, incremental JSONL storage, screenshots, and Excel reporting.
